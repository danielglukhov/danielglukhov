*Мобильное приложение для игроков в настольные RPG.*

DiceKeep — независимый проект, который позволяет мне объединить опыт UX/UI-дизайнера и Flutter-разработчика в одном продукте.

Работая над ним, я самостоятельно прохожу все этапы разработки мобильного приложения: исследование идеи, проектирование пользовательского опыта, создание дизайн-системы, разработку, тестирование и подготовку к публичному релизу.
